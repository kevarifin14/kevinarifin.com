Next.js supports absolute imports out-of-the-box by setting the `baseUrl` in the `tsconfig.json`. 

```json
// tsconfig.json
{
  "compilerOptions": {
    "baseUrl": "."
    // ...
  }
}
```

Now, if you have a `Button` component in `components/Button/index.tsx`, you can import the component with an absolute import.

```tsx
import { Button } from 'components/Button';

// ...
```
