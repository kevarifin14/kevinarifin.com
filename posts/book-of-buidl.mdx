This is a massive post consolidates all the blog posts that are part of a full-stack Next.js application to spin up a startup website.

You can start with the default Next.js repo and work through the tutorials

```bash
yarn create next-app --typescript my-app-name
```

Or you can just skip ahead and download the starter repo

```bash
yarn create next-app --example https://github.com/kevarifin14/buidling-blocks my-app-name
```
